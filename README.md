# صيف وصحة - مساعد

منصة Django SSR عربية لخدمة زوار عسير: خطة صحية يومية، فعاليات، مواقع قريبة، رسائل توعوية، مكتبة تحميلات، الدكتور مساعد، جواز صحة عسير، QR، وتحدي بطل الصحة.

## تشغيل محلي سريع

```powershell
.\venv\Scripts\python.exe manage.py migrate
.\venv\Scripts\python.exe manage.py runserver
```

## Docker Compose

```powershell
docker compose up --build
```

لوحة الإدارة المخصصة: `/admin/`  
لوحة Django الرسمية: `/django-admin/`
